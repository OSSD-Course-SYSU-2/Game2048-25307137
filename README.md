这是一个2048游戏。
基本玩法：
棋盘：一个4*4的方格，初始会有两到四个数字方块。
操作：用手指滑动屏幕或用屏幕下方的按钮，屏幕上所有方块会朝该方向移动。
合并规则：移动时；两个数字相同的方块碰到一起，会合并成它们的和。每次合并后，得分增加合并后的数值。
结束条件：
胜利：成功合成一个2048方块（还可以继续游玩挑战更高粉数）
失败：所有格子填满，且没有任何相邻方块数字相同。
游戏基本功能：
可以点“新游戏重新开始一局新游戏”
会记录当前分数和游玩历史最高分数。
